Check out my portfolio here: **https://prasanna-godiyal-portfolio.vercel.app/**

## ✨ Features

* 👨‍💻 About Me
* 🛠️ Technical Skills
* 🚀 Featured Projects
* 🎓 Education
* 📄 View & Download Resume
* 📬 Contact Information
* 📱 Fully Responsive Design

## 🛠️ Tech Stack

* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Vite

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/PrasannaGodiyal3903/Prasanna-Godiyal--Portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
